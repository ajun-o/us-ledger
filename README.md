# Us Ledger — 双人记账 PWA

情侣/伴侣共同管理财务的记账应用，支持三视角切换、AA 计算、预算管理、存钱计划等功能。

## 技术栈

React 19 + TypeScript + Vite + Supabase

## 功能

### 记账核心
- 金额校验 + 分类必选 + 抖动提示
- 离线保存队列（断网入队，恢复自动同步）
- 账户选择器（底部弹出含余额）
- 日期时间选择器（不可选未来）
- 备注 @提及伴侣
- 编辑预填充 + 删除 3 秒可撤销

### 主页
- 月份选择器（年/月滚轮，不能选未来）
- 三视角卡片左右滑动 + 金额滚动动画
- 金额显隐持久化（跨视角/月份）
- 账单行点击右侧滑入详情
- 下拉刷新（>60px 触发 + 3 秒冷却）
- 空状态点击快捷记账

### 账单页
- 筛选抽屉（成员/分类/金额/账户）
- 搜索防抖 300ms + 匹配高亮
- 收支切换标签
- 排序切换（时间/金额/分类）
- 长按批量模式 + 左滑编辑删除
- 统计栏（月结余 + 日均支出 + 计算说明浮层）

### 报表页
- 日历日期点击展开/收起当日账单
- 日历左右滑动切月 + 今日高亮
- 图表视图（饼图/面积/柱状）
- 图表下钻 + 二级分类
- 图表分享（保存图片 + 系统分享 + Logo 日期水印）
- 统计栏（月支出 + 日均支出 + 计算详情浮层）

### 资产管理
- 账户列表 + 添加/编辑/删除
- 余额调整 + 调整记录
- 删除账户时关联账单迁移

### 个人中心
- 用户资料页（头像/昵称编辑 + 记账天数 + 解绑二次确认）
- 收支分类管理（新建/编辑/删除 + 系统分类 + 账单迁移）
- 预算设置（总预算 + 分类预算 + 预警阈值 + 超预算提醒）
- 多账本管理（新建/编辑/默认/归档/删除 + 账单迁移）
- 标签管理（新建/编辑/删除 + 颜色选择 + 标签账单查看）
- 账单报告（周期选择 + 环形图 + TOP5 + 成员对比 + 趋势图 + 趣味统计 + 分享）
- 定时记账（启用/禁用 + 周期选择 + 分类 + 成员 + 账户设置）
- 存钱计划（目标设定 + 进度条 + 存入记录 + 达成庆祝动画）
- 购物清单（添加/编辑/删除 + 购买勾选 + 预估/实际价格 + 分类 + 统计）

### 辅助工具
- 汇率换算（12 币种 + 币种选择器 + 双向兑换 + 汇率刷新）
- 小工具箱（折扣计算 + AA 分摊 + 分期计算 + 预设快捷按钮）
- 个性化设置（5 主题色 + 3 档字体 + 实时预览 + 导航栏联动）

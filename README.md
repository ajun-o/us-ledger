# Us Ledger

## 这是什么
双人记账 PWA，情侣/伴侣共同管理财务，支持三视角切换、AA 计算、预算管理。

## 项目类型
Web 前端（PWA）

## 技术栈
React 19 + TypeScript + Vite + Supabase

## 为什么做
练习全栈 PWA 开发，模仿真实记账 App 的交互体验。

## 状态
🟢 进行中

## 怎么跑起来
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
